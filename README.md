# Homesorted - Hyperlocal Service Marketplace

A modern, professional landing page for Homesorted, a startup mobile application that connects local skilled workers with customers instantly.

## 🚀 Features

- **Modern UI/UX**: Clean, premium design with gradients, animations, and glassmorphism effects
- **Responsive Design**: Mobile-first approach that works perfectly on all devices
- **Smooth Animations**: Framer Motion powered animations and transitions
- **Fast Loading**: Optimized for performance with modern web technologies
- **SEO Optimized**: Built with best practices for search engine visibility

## 🛠️ Tech Stack

- **React 18** - Modern React with hooks and functional components
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library for React
- **Lucide React** - Beautiful icons

## 📱 Sections

1. **Hero** - Eye-catching introduction with animated phone mockup
2. **How It Works** - 4-step process explanation
3. **Services** - Service categories with icons

5. **Worker Verification** - Trust-building verification process
6. **App Preview** - Realistic mobile app screens
7. **Testimonials** - Customer and worker reviews
8. **For Workers** - Dedicated section for service providers
9. **Statistics** - Animated counters and metrics
10. **FAQ** - Accordion-style frequently asked questions
11. **Final CTA** - Strong conversion section
12. **Footer** - Comprehensive footer with links

## 🏃‍♂️ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd homesorted-landing
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```
